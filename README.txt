*Суть проблемы
Организация занимается оптовой торговлей практически со всеми регионами России, ведя базу данных в 1С.
Прайс-лист представлен в формате xls, который рассылается клиентам еженедельно по электронной почте.
Получив свежий прайс, клиенты в строке с интересующим товаром и в отдельной колонке указывают необходимое
количество для заказа товара. После этого,  клиент кликает на кнопку (сам файл с макросами), после чего
происходит формирование отдельного файла «лист заказа» и
который в свою очередь отправляется менеджеру для о последующего оформления в 1С документов.


Цель: уйти от формирования файла и рассылки прайс-лист для большинства клиентов, сделав для них личный кабинет.
